# Sistema Adoção Banco de Dados

Projeto desenvolvido para o Módulo 3 — Banco de Dados e Controle de Versão.

O projeto apresenta a modelagem e implementação de um banco de dados para um sistema web de adoção de animais.

## Tecnologias utilizadas

- MySQL
- SQL
- Git
- GitHub

## Estrutura do projeto

database.sql → criação das tabelas e relacionamentos

consultas.sql → comandos de manipulação de dados

docs → imagens do diagrama e consultas SQL

## Funcionalidades

- Modelagem do banco de dados
- Criação de tabelas
- Inserção de dados
- Atualização de registros
- Remoção de dados
- Consultas SQL
- Controle de versão utilizando GitHub
